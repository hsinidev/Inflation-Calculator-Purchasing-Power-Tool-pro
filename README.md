## ğŸ¯ About The Project

The **Inflation Calculator** is more than just a calculation utility; it is an educational platform wrapped in a stunning, immersive galaxy-themed interface. It addresses a critical need in personal finance: understanding the "real" value of money.

By leveraging historical Consumer Price Index (CPI) data, this tool allows users to input an amount of money from a past year and instantly see its equivalent value in a different year. This visualization of purchasing power loss is essential for investors, savers, and anyone planning for their financial future.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v16+)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/hsinidev/inflation-calculator.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd inflation-calculator
    ```

3.  **Install dependencies:**
    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
